Данная работа посвящена численному моделированию движения автомобиля с нелинейным демпфером под действием дорожного профиля. Модель описывается системой обыкновенных дифференциальных уравнений второго порядка, которая решается методом Рунге-Кутты 4-го порядка.
Основные задачи:
-Расчёт параметров системы (масса, жёсткость пружины, коэффициенты демпфирования)
-Численное интегрирование уравнений движения
-Оценка погрешности метода Рунге-Ромберга
-Анализ чувствительности решения к погрешности исходных данных
-Сравнение трёх типов демпферов (линейный, слабо- и сильнонелинейный)
Библиотеки, используемые в программе:
- NumPy (np.linalg.solve() - решение системы линейных уравнений для расчёта r0; np.linspace(), np.zeros(), np.gradient())
- Matplotlib (plt.plot() - графики перемещения, скорости, ускорения; plt.subplot() - объединение 4 графиков в одном окне)
- SciPy (численное интегрирование, решение нелинейных уравнений, обработка сигналов)
